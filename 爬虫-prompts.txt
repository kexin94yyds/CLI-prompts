作业 - 连接并验证（输入密码：Linux@123） ssh -o PreferredAuthentications=password -o PubkeyAuthentication=no linux_7230264145@43.139.230.196 登录后执行： clear; echo "连接验证"; whoami; sudo whoami 常见报错一键处理 - 被公钥拒绝（Permission denied (publickey)） 已用强制密码参数，仍失败则重试上面命令并确认密码大小写正确。 - Host key 报错（Host key verification failed） ssh-keygen -R 43.139.230.196 && ssh -o PreferredAuthentications=password -o PubkeyAuthentication=no linux_7230264145@43.139.230.196 - 端口不是 22（如自定义端口 2222） ssh -p 2222 -o PreferredAuthentications=password -o PubkeyAuthentication=no linux_7230264145@43.139.230.196 一次性加速（可选，做一次就行） - 建立别名后以后直接 ssh mylinux mkdir -p ~/.ssh && chmod 700 ~/.ssh && { echo "Host mylinux"; echo " HostName 43.139.230.196"; echo " User linux_7230264145"; echo " PreferredAuthentications password"; echo " PubkeyAuthentication no"; } >> ~/.ssh/config && chmod 600 ~/.ssh/config 以后连接： ssh mylinux
